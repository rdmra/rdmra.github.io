  1. Harga sewa berbeda tergantung jumlah murid. Hal ini dikarenakan semakin banyak siswa, semakin banyak memakan resource hosting.

Pertanyaan Umum

  • Bukankah RDM itu gratis? Kenapa harus sewa Hosting?

    Ya, RDM itu gratis. Kami tidak memperjual belikan RDM. Kami hanya menyewakan hosting (server) untuk data RDM. Ibaratnya seperti ini, RDM itu Tenda. Bapak/Ibu dikasih tenda gratis oleh pemerintah, tapi ketika mau camping, tetap harus mencari lokasi terbaik dan sewa tempat, itu perumpamaan hosting.

  • Apakah harus install sendiri?

    Tidak usah. bapak ibu tidak usah dipusingkan dengan instal cPanel, database dan sejenisnya. Kami yang instalkan. Bapak/Ibu tinggal registrasi menggunakan NSM dan TOKEN.

  • Setelah saya Order, apa yang harus saya lakukan?

    Setelah order dan mendapatkan link /alamat web RDM RA Bapak/Ibu ada beberapa langkah diantaranya :
    - Registrasi RDM menggunakan TOKEN supaya tersambung ke server pusat
    - Setting RDM (pengaturan, input guru, syncron siswa dll)
    - Guru bisa login dengan akun masing-masing dan mengisi nilai
    - Rapor siap diprint
    untuk lebih jelas cara registrasi, setting, input nilai, kunjungi Halaman Tutorial DISINI.

  • Dari mana saya mendapat Token?

    Token didapatkan dari Operator Kabupaten. Atau bisa kami bantu, silahkan hubungi kami untuk mendapat Token.

  • Apakah data Sekolah saya aman?

    Kami melakukan backup database secara berkala, jadi Insya Allah data aman. Kemudian Bapak/Ibu proktor juga harus melakukan backup database secara berkala.

  • Kapan saya harus melakukan perpanjangan sewa hosting?

    Tahun depan. Karena sewa hosting berlaku selama 1 tahun (2 semester)

  • Apakah saya bisa tanya-tanya sebelum Order?

    Selama kami tidak ada halangan, Insya Allah konsultasi Bapak/Ibu akan kami jawab.
